💻 El Concepto de Límite en Algoritmia

En ingeniería de software, un límite nos ayuda a entender el comportamiento de un algoritmo a medida que el tamaño de la entrada (n) crece. Esto es la base de la Complejidad Asintótica (Big O).

limn → ∞ f(n) / g(n) = C

Si este límite es una constante, decimos que f(n) es O(g(n)). Nos permite predecir si un software colapsará con millones de usuarios.

🌐 Latencia de Red y Timeouts

Analizamos el límite del tiempo de respuesta (f(t)) cuando el tráfico (t) tiende a la capacidad máxima del servidor. Si el límite tiende a infinito, se requiere un balanceador de carga.
